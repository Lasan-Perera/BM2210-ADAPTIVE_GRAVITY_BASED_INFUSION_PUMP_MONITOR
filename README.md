A smart IV (Intravenous) fluid regulator system designed to monitor and control the drip rate in medical settings. The project combines embedded electronics, sensors, and software to ensure accurate and safe fluid delivery to patients.

## Features
- Real-time monitoring of IV drip rate (drops/min and mL)
- Adjustable drip rate with doctor-friendly interface
- Automatic alerts for rate deviations
- Data display on local screen and optional web/mobile monitoring
- Power-saving mode to prolong battery life
- Software keypad lock to prevent accidental changes
- Two-step verification for web application access
